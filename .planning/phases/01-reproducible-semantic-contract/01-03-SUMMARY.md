## Decisions Made

- Constants retain their public uppercase names; functions and class members use mapped camelCase while classes/types remain PascalCase.
- Pure constants, semantic types/helpers, codecs, and all nine public register APIs are Phase 1-owned; transport/detection runtime classes are Phase 2, write-only `WriteSafetyResult` is Phase 3, and all web symbols are exclusively Phase 4/`./web`.
- `WriteClass` remains Phase 1-owned as a frozen semantic domain value required by immutable register definitions; no write operation is implemented or claimed.
- A class row carries a closed representation and member-naming policy joined to the complete Python fact inventory, avoiding copied defaults/signatures and preventing a second class-fact authority.
- `--release` is intentionally red during Phase 1 because every row remains `planned`; Plan 01-09 alone may promote fully implemented and evidenced Phase-1 rows.
